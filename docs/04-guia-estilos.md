## Guía de estilos

La guía recoge la paleta de colores, las tipografías (**EB Garamond** para titulares, **Inter** para el cuerpo) y el sistema de espaciado basado en una escala de 4 px.

## Componentes reutilizables

Botones, inputs, badges de estado, tarjetas, modales, tablas, toasts y navegación. Cada componente tiene definidos sus estados (normal, hover, focus, disabled, error) y está trasladado directamente a `frontend/src/styles/05-components/`.
